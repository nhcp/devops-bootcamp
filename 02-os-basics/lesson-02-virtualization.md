# Module 2, Lesson 02: Introduction to Virtualization & Virtual Machines

## 🎯 Objective
Understand how virtualization technology allows for isolated environments and efficient resource management.

## 💡 What is Virtualization?
Virtualization is the process of creating a software-based (virtual) version of something, such as a server or network. It allows one physical machine to run multiple independent systems simultaneously.

## 💻 Virtual Machines (VMs)
* **Definition**: A digital version of a physical computer that runs its own Guest Operating System.
* **The Hypervisor**: The software layer that creates and manages VMs by distributing physical resources (CPU, RAM) from the Host machine to the Guest VMs.

## 🛠️ Why DevOps uses Virtualization
* **Isolation**: Test code in safe, isolated environments.
* **Cost Efficiency**: Maximize hardware by running many virtual servers on one physical machine.
* **Standardization**: Containers and VMs are the modern standard for deploying applications.
