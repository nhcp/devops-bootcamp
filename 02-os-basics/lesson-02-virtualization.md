# Module 2, Lesson 02: Introduction to Virtualization & Virtual Machines

## 💡 What is Virtualization?
Virtualization is the process of creating a software-based (virtual) version of something, like a server. It allows one physical machine to run multiple independent systems.

## 💻 Virtual Machines (VMs)
* **Definition**: A digital version of a physical computer.
* **The Hypervisor**: The software layer that manages VMs by distributing physical resources (CPU, RAM) from the Host machine to the Guest VMs.

## 🛠️ Why DevOps uses Virtualization
* **Isolation**: Test code in safe, isolated environments.
* **Efficiency**: Run many virtual servers on one physical machine.
