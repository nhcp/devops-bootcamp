# Module 2, Lesson 01: Introduction to Operating Systems

## 🎯 Objective
Understand the fundamentals of Operating Systems (OS) as a prerequisite for managing infrastructure.

## 🏗️ What is an Operating System?
An OS is the master software that manages hardware (CPU, RAM, Storage) and provides an environment for apps to run.

## 🧠 Core OS Architecture
* **The Kernel**: The core "brain" of the OS that manages system resources.
* **User Space**: Where user applications run safely, isolated from the hardware.
* **CLI vs. GUI**: DevOps work is done in the CLI for speed and automation.
