# Module 2, Lesson 01: Introduction to Operating Systems

## 🎯 Objective
Understand the fundamentals of Operating Systems (OS) as a prerequisite for managing infrastructure.

## 🏗️ What is an Operating System?
An OS is the master software that manages a computer's hardware—including the CPU, RAM, and Storage—and provides an environment for applications to run.

## 🧠 Core OS Architecture
* **The Kernel**: The core "brain" of the OS. It sits between the hardware and software, managing system resources.
* **User Space**: The isolated area where user applications run safely, separated from the hardware for system stability.
* **CLI vs. GUI**: DevOps work is done primarily in the Command Line Interface (CLI) because it is lightweight and efficient for automation.
